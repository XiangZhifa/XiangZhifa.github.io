user指令

  1. user用于配置运行 Nginx worker进程 的 用户 和 用户组
  2. 语法：user [user] [group] ( 不配置用户组的情况下，group默认值与user保持一致 )
  3. user指令的默认值 nobody

案例

!!!tips!!!

要将Nginx需要访问的相关文件，存放在user指令配置的用户对应的文件夹下，否则访问时会报403
