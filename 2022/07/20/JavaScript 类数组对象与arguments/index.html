类数组对象的应用

Arguments 对象就是一个类数组对象。

在客户端 JavaScript 中，一些 DOM 方法 ( document.getElementsByTagName()、document.getElementsByClassName() 等) 也返回类数组对象。

Arguments对象

Arguments 对象只定义在函数体中，包括了函数的参数和其他属性。在函数体中，arguments 指代该函数的 Arguments 对象。

Arguments对象 length属性

Arguments对象 的length属性，表示实参的长度。

Arguments对象 callee属性

Arguments 对象的 callee 属性，通过它可以调用函数自身。

arguments 和对应参数的绑定

传入的参数，实参 和 arguments 的值会绑定；当没有传入时，实参与 arguments 值不会绑定。

除此之外，以上是在非严格模式下，如果是在严格模式下，实参 和 arguments 是不会绑定的。
